¡Bienvenido!

Esta es una aplicación para calcular el presupuesto de una página web, la cual empezará con tres checkboxes mediante los cuales el usuario podrá decidir si desea obtener el presupuesto de:
  • Una página web (500 €)
  • Una campaña SEO (300 €)
  • Una campaña de publicidad (200 €)

En función de las opciones que marque, se mostrará un precio distinto.

Pulsa el botón iniciar para acceder a nuestra calculadora de presupuestos.
